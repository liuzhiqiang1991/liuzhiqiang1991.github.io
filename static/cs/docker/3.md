#### 常见命令

```
    docker pull nginx

    docker images

    docker save --help

    docker save -o nginx.tar nginx:latest

    docker rmi nginx:latest

    docker load -i nginx.tar

    docker run -d --name nginx -p 80:80 nginx

    docker ps

    docker stop nginx

    docker ps -a

    docker start nginx

    查看日志： docker logs nginx -f

    进入容器： docker exec -it nginx bash

    删除容器： docker rm mysql -f
    或者：
              docker stop mysql
              docker rm mysql
```

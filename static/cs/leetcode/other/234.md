#### 234. 回文链表
```
    class Solution {
        public boolean isPalindrome(ListNode head) {
            Deque<Integer> queue = new ArrayDeque<>();
            while(head != null) {
                queue.addLast(head.val);
                head = head.next;
            }
            while(queue.size() > 0) {
                int a = queue.removeFirst();
                if(queue.size() > 0) {
                    int b = queue.removeLast();
                    if(a != b) return false;
                }
            }
            return true;
        }
    }
```

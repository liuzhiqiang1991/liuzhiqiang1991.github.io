#### 169. 多数元素
```
    class Solution {
        public int majorityElement(int[] nums) {
            int candNum = nums[0];
            int count = 1;
            for(int i = 1; i <= nums.length - 1; i++) {
                if(nums[i] == candNum) count++;
                else count--;
                if(count == 0) {
                    candNum = nums[i + 1];
                    count = 0;
                }
            }
            return candNum;
        }
    }
```

### 144. 二叉树的前序遍历

```
    class Solution {
        public List<Integer> preorderTraversal(TreeNode root) {
            List<Integer> res = new ArrayList<>();
            Stack<TreeNode> stack = new Stack<>();
            while(stack.size() > 0 || root != null) {
                while(root != null) {
                    res.add(root.val);
                    stack.push(root);
                    root = root.left;
                }
                if(stack.size() > 0) {
                    root = stack.pop();
                    root = root.right;
                }
            }
            return res;
        }
    }
```

### 索引库的CRUD
#### 1、创建索引库和映射
```
    # PUT /test
    {
      "mappings": {
        "properties": {
          "info":{
            "type": "text",
            "analyzer": "ik_smart"
          },
          "email":{
            "type": "keyword",
            "index": "false"
          },
          "name":{
            "properties": {
              "firstName": {
                "type": "keyword"
              }
            }
          }
        }
      }
    }
```


#### 2、查询索引库： GET /test

#### 3. 修改索引库
```
    PUT /test/_mapping
    {
      "properties": {
        "age":{
          "type": "integer"
        }
      }
    }
```

#### 4. 删除索引库： DELETE /test

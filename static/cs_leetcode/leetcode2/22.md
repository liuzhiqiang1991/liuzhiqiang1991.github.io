### 22.括号生成

```
    class Solution {
        public List<String> generateParenthesis(int n) {
            Set<String> set = new HashSet<>();
            set.add("");
            for(int i = 1; i <= n; i++) {
                Set<String> set2 = new HashSet<>();
                for(String s : set) {
                    for(int j = 0; j <= s.length(); j++) {
                        String left = s.substring(0, j);
                        String right = s.substring(j);
                        set2.add(left + "()" + right);
                    }
                }
                set = set2;
            }
            set.remove("");
            return new ArrayList<>(set);
        }
    }
```

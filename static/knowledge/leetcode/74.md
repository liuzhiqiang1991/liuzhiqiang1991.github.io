#### 74.搜索二维矩阵
```
    class Solution {
        public boolean searchMatrix(int[][] matrix, int target) {
            int m = matrix.length;
            int n = matrix[0].length;
            int left = 0;
            int right = m * n - 1;
            while(left <= right) {
                int mid = (left + right) / 2;
                int x = matrix[mid / n][mid % n];
                if(x == target) {
                    return true;
                } else if(x > target) {
                    right = mid - 1;
                } else {
                    left = mid + 1;
                }
            }
            return false;
        }
    }
```
